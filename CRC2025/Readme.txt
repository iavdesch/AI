Readme
======

The supporting datafiles are related to the conference paper "The Effect of Shrinkage Prior to Testing on the Prediction of the Crack Widths in Reinforced Concrete Elements".

Related to this research, the following files are relevant:
Research/CRC2025/Data/ (containing the supporting data files)
Research/CRC2025/Notebooks/ (containing the source code to generate the result table in the paper)
Research/Crack_width_formulas/ (containing the source code for data selection, processing (derivation of variables) and filtering, needed to evaluate the formulas for crack width)
